Назначение:

Отображение информации о загруженных данных, а именно о количестве загруженных в СУБД записей в десятиминутных интервалах.


Файлы проекта:

monitor.py	- основной файл проекта (скрипт)
config.ini	- конфигурационные параметры, необходимые для работы скрипта
static		- папка со статическими файлами
templates	- папка с html-шаблонами проекта (jinja)


Дополнительные файлы:

requirements.txt
Список используемых пакетов.


Замечания:

В качестве СУБД используется MySQL Server 5.7.

В проекте пока отсутсвует авторизация и отображение информации о логгах (не требовались в ТЗ)
Не удалось настроить корректную работу некоторых элементов графика на веб-страницу (Highcharts): не отображается элемент subtitle, некорректно отражаются значения на оси X (хотя метки графика отражаются правильно).
В графике отражаются данные за 1 день, а не за 7 (как указано в ТЗ), т.к. при количестве 7 дней график ничего не отражает - видимо превышено количество допустимых значений. Максимальное количество отражаемых дней в моем случае оказалось 3, для проекта я, в конечном счете, оставил 1 день.
